# Представление проекта:
```
|__ Domain (Слой ядра и правил)
|   |__ models
|   |__ repositories
|
|__ Application (Слой бизнес-логики и правил валидации)
|   |__ validator
|
|__ API (Слой взаимодействия с внешним миром)
|   |__ controller
|   |__ view
```
- Domain: Описано «что» мы храним (сущности) и «как» мы хотим ими управлять (интерфейсы).
- Application: Реализован «фильтр» безопасности.
- API: Написаны контроллеры, которые принимают сырые данные, проверяют их и отдают ответы.
